# Invoice CRM & Payment Tracking System

Monorepo with separate **frontend** (React + Vite) and **backend** (Node.js + Express + Sequelize + MySQL).

## Quick start

### Backend

```bash
cd backend
cp .env.example .env
npm install
npm run dev
```

### Frontend

```bash
cd frontend
cp .env.example .env
npm install
npm run dev
```

Make sure your MySQL database settings are configured in `backend/.env`.
